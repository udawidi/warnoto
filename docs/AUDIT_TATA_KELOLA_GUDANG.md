### Daftar Eviden Wajib Dilampirkan

#### A. Dokumen Proses Bisnis
- [ ] 7 (tujuh) Dokumen Proses Bisnis (flowchart) masing-masing **bertanda tangan General Manager**
- [ ] Setiap dokumen mencakup alur: Penerimaan → Penyimpanan → Pengeluaran → Retur

#### B. Siklus Lengkap Eviden Penerimaan (2 Sampling Material)
Untuk **masing-masing** dari 2 sampling material, wajib dilampirkan:
- [ ] Surat Jalan (SJ) dari vendor
- [ ] Kontrak / Surat Perintah Barang (SPB)
- [ ] Form PO (Purchase Order)
- [ ] Form Pemeriksaan **TUG 4** (beserta catatan pemeriksaan fisik)
- [ ] Form Penerimaan **TUG 3** (tanda terima resmi)
- [ ] Kartu Gantung **TUG 2** (label identitas barang)
- [ ] **Foto Material** (kondisi fisik saat diterima)

> ⚠️ **Catatan**: Eviden harus lengkap untuk kedua sampling. Ketidaklengkapan pada salah satu item mengakibatkan temuan non-konformitas.

#### C. Surat Pemberitahuan Vendor & Dokumen Pendukung
- [ ] Surat Pemberitahuan Vendor (dikirim >7 hari sebelum rencana pengiriman)
- [ ] Tanda Terima / Berita Acara (BA) penerimaan surat pemberitahuan
- [ ] Dokumen cek selisih hari pengiriman (perbandingan tanggal surat vs tanggal aktual)

#### D. Siklus Lengkap Eviden Pengeluaran (2 Sampling Material)
Untuk **masing-masing** dari 2 sampling material, wajib dilampirkan:
- [ ] Surat Permintaan dari User/Pemohon
- [ ] Kontrak Eksternal *(jika pengeluaran terkait pekerjaan eksternal)*
- [ ] Reservation Slip / Work Order (WO) SAP
- [ ] Slip Pengeluaran **TUG 9**
- [ ] Kartu Gantung **TUG 2** (update stok pengeluaran)
- [ ] **Foto Proses** pengeluaran/penyerahan material

#### E. Foto Penataan Rak
- [ ] Foto penataan rak **melintang** (bukan longitudinal)
- [ ] Terlihat **list/daftar material** pada setiap rak
- [ ] **Kartu gantung** menempel pada masing-masing lokasi rak

#### F. Referensi Penyimpanan Khusus
Untuk **3 sampling material** (MTU / Isolator / Kabel):
- [ ] **Manual Book** / Buku Kajian Umur Maksimum (dokumen referensi)
- [ ] **Foto kesesuaian fisik** penyimpanan material dengan panduan manual book
- [ ] Kondisi penyimpanan: suhu, kelembaban, pencahayaan, jarak dari material lain
